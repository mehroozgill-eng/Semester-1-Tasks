# Week 3 Tasks



Basic Arithmetic Operations in C



&#x20;Description



This C program performs basic arithmetic operations on two integers entered by the user.

The program takes two integer inputs and calculates:



\* Sum

\* Difference

\* Product

\* Quotient



It then displays the results on the screen.







&#x20;Features



\* Takes \*\*two integer inputs\*\* from the user.

\* Performs \*\*four basic arithmetic operations\*\*:



&#x20; \* Addition

&#x20; \* Subtraction

&#x20; \* Multiplication

&#x20; \* Division

\* Displays results using formatted output.







Technologies Used



\* C Programming Language

\* Standard Input/Output Library (`stdio.h`)



&#x20;Program Logic



1\. The program declares two integer variables:



&#x20;  \* `num1`

&#x20;  \* `num2`



2\. The user enters the \*\*first integer\*\*, which is stored in `num1`.



3\. The user enters the \*\*second integer\*\*, which is stored in `num2`.



4\. The program performs the following calculations:



&#x20;  \* \*\*Sum\*\* → `num1 + num2`

&#x20;  \* \*\*Difference\*\* → `num1 - num2`

&#x20;  \* \*\*Product\*\* → `num1 \* num2`

&#x20;  \* \*\*Quotient\*\* → `num1 / num2`



5\. Each result is printed using `printf()`.
